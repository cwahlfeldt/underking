# Underking

A hex-grid turn-based game built with Rust and [Bevy](https://bevyengine.org/) 0.18.

## Running

```bash
cargo run
```

## Controls

- **Click** a hex tile to move the player
- **Hover** over an enemy to see its attack range
- **Backtick (`)** toggles the debug overlay (grid state, entity info, hex coordinates)

## Tests

```bash
cargo test
```
